# Extented ACL configuration (CCNA)

Tool:
Cisco Packet Tracer

Concepts Practiced:
- define ACLs with diffrent ACEs
- apply ACL to interfaces as inbound & outbound
- Implicit deny concept

- 

Commands Used:

- ip access-list extended {ACL_NAME}
- {DENY | PERMIT} {PROTOCOL} {SRC_IP} {DST_IP}
- Permit ip any any
- ip access-group {ACL_NAME} {IN | OUT}
  

Result:

 Successfully block some packet transfering to another part of the network.
